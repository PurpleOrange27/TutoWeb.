Bab 4

CSS Frameworks

Mempercepat proses desain website menggunakan pustaka kode (library) siap pakai.

4.1 Pengenalan CSS Frameworks

CSS Framework adalah kumpulan kode CSS yang sudah disiapkan agar kita tidak perlu mendesain elemen dasar dari nol.

Keuntungan: Desain lebih cepat, sudah responsif (mobile-friendly), dan standar industri.

4.2 Bootstrap 5: Component Based

Bootstrap adalah framework paling populer. Kita hanya perlu menambahkan nama class tertentu ke HTML.

Grid System

<div class="row">
  <div class="col-6">50%</div>
  <div class="col-6">50%</div>
</div>

Komponen Tombol

class="btn btn-primary"

4.3 Tailwind CSS: Utility-First

Tailwind memberikan class-class kecil (utility) untuk membangun desain sesuka hati.

tailwind-button.html
<button class="bg-blue-500 text-white font-bold py-2 px-4 rounded">
  Tombol Saya
</button>

4.4 Perbandingan Frameworks

FiturBootstrap 5Tailwind CSS
PendekatanKomponen Siap Pakai (Instan)Utility Classes (Fleksibel)
CustomisasiAgak sulit (Override CSS)Sangat mudah
