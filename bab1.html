Bab 1

Pengenalan Pemrograman Web

Memahami fondasi dasar bagaimana internet dan website bekerja.

1.1 Definisi Pemrograman Web

Pemrograman web adalah proses pembuatan instruksi-instruksi untuk menciptakan program/aplikasi yang dapat diakses melalui internet menggunakan browser.

1.2 Website Statis vs Dinamis

Fitur
Statis (HTML/CSS)
Dinamis (PHP/JS/DB)
Konten
Tetap, harus edit kode manual
Berubah otomatis (dari database)
Interaksi
Hanya membaca (Read Only)
Bisa login, komentar, transaksi
Contoh
Company Profile sederhana
Facebook, Tokopedia

1.4 Tools & Lingkungan Pengembangan

  • Code Editor: Visual Studio Code (VS Code) - Paling populer.
  • Web Browser: Google Chrome atau Microsoft Edge (untuk debugging).
  • Extension: Live Server (agar auto-refresh saat save).
