CSS3 ‚Äì Styling dan Layout

Mengubah kerangka HTML menjadi halaman web yang indah, responsif, dan interaktif.

3.1 Pengenalan CSS3

CSS (Cascading Style Sheets) berfungsi memisahkan konten (HTML) dengan desain (Visual). CSS3 adalah versi terbaru yang mendukung animasi, gradasi, dan tata letak modern.

Tanpa CSS, semua website akan terlihat seperti dokumen Word biasa (hitam putih).

3.2 Selektor, Properti, dan Nilai

Rumus dasar penulisan CSS:

selektor {
    properti: nilai;
}
h1 { color: blue; }

3.5 Layout: Flexbox & Grid

Teknik modern untuk menyusun tata letak website agar rapi.

Flexbox (1 Dimensi)

.container {
  display: flex;
  justify-content: center;
}

Grid (2 Dimensi)

.galeri {
  display: grid;
  grid-template-columns: 1fr 1fr;
}
