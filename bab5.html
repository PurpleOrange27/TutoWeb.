Bab 5

JavaScript Modern (ES6+)

Membuat website menjadi interaktif, dinamis, dan cerdas.

5.1 Pengenalan JavaScript

JavaScript (JS) adalah otot dan otak website.

Kita menggunakan standar ES6 (ECMAScript 2015+) yang lebih ringkas dan powerful.

5.2 Variabel, Tipe Data, & Operator

Gunakan let dan const, tinggalkan var.

script.js
const umur = 21; // Tetap
let skor = 0;    // Bisa berubah
const hobi = ["Koding", "Gaming"];

5.3 Struktur Kontrol

If / Else

if (nilai >= 75) {
  console.log("Lulus");
}

Looping

for (let i = 0; i < 5; i++) {
  console.log(i);
}
