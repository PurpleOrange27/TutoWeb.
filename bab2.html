HTML5 ‚Äì Struktur Dasar Website

Mempelajari elemen-elemen pembangun halaman web modern dari teks, tabel, hingga multimedia.

2.1 Pengenalan HTML5

HTML5 adalah versi terbaru dari HTML yang mendukung elemen semantik, audio, video, dan grafis tanpa perlu plugin tambahan.

Browser modern (Chrome, Edge, Firefox) sudah mendukung penuh standar HTML5.

2.2 Struktur Dasar Dokumen HTML

Setiap file HTML harus memiliki kerangka standar berikut:

struktur.html
<!DOCTYPE html>
<html lang="id">
  <head>
    <meta charset="UTF-8">
    <title>Judul Halaman</title>
  </head>
  <body>
    <!-- Konten website ditulis di sini -->
  </body>
</html>

2.4 Hyperlink dan Navigasi

Link menghubungkan satu halaman ke halaman lain menggunakan tag <a>.

<!-- Link ke website luar -->
<a href="https://google.com" target="_blank">Buka Google</a>

<!-- Link ke halaman lokal -->
<a href="about.html">Tentang Kami</a>
